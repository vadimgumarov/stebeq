# Stack Documentation

This folder contains technology stack documentation.

## Contents

- Technology choices and rationale
- Database setup and migrations
- Third-party integrations
- Development environment setup
- **Research decisions** - Tech stack research for complex features

## Research Documents

For complex features requiring tech stack decisions, create:
```
docs/stack/research-[feature-name].md
```

Use `research-TEMPLATE.md` as starting point.

**When to create research doc:**
- Choosing between libraries/frameworks
- Architectural decisions with trade-offs
- New integrations requiring evaluation
- Performance optimization approaches

## When to Update

- New technology added
- Database schema changes
- New integrations configured
- Tech stack decisions made (add research doc)
